Пользовательское соглашение приложения Dangeon and Dangeon


Настоящий документ «Пользовательское соглашение» представляет собой предложение Лотонова Данилы, далее - Разработчика, заключить договор на изложенных ниже условиях Соглашения.


Список понятий


Приложение - программное обеспечение, предоставляемое Разработчиком Пользователю.


Пользователь - лицо, использующее какие-либо функции или части Приложения.


Разработчик - лицо, собственностью которого является Приложение.


Обязанности Разработчика


1. Разработчик обязуется предоставлять полный доступ к пользованию, модификации Приложения.
2. Разработчик обязуется распространять копии Приложения безвозмездно.


Ограничения Разработчика


1. Разработчик не имеет права изменять пользовательское соглашение без уведомления пользователей Приложения.
2. Разработчик не имеет права внедрять в Приложение какие-либо функции, наносящие вред аппаратному или программному обеспечению Пользователя, не относящиеся к Приложению.


Права Разработчика


1. Разработчик имеет право ограничить доступ к Приложению для физических лиц, предварительно уведомив их об ограничении.
2. Разработчик имеет право изменять функционал Приложения без уведомления пользователя.
3. Разработчик не несёт ответственности за материальный, моральный, культурный, физический, психический вред, нанесённый Пользователю во время пользования Приложением.


Обязанности Пользователя


1. Пользователь обязан соблюдать актуальное лицензионное соглашение.


Ограничения Пользователя


1. Пользователь не имеет права на платное распространение Приложения в любом виде.
2. Пользователь не имеет права выдавать модифицированную копию Приложения за программное обеспечение, находящееся в его собственности.


Права Пользователя


1. Пользователь вправе без каких-либо ограничений модифицировать копию Приложения, предоставленную ему Разработчиком.
